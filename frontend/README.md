# GPX Helper Frontend

## Prerequisites

- Node.js 18+
- npm

## Install

```sh
npm install
```

## Run the dev server

```sh
npm run dev
```

The app will be available at <http://localhost:4173>.

## Build for production

```sh
npm run build
```

## Run tests

```sh
npm run test
```

## Run tests in watch mode

```sh
npm run test:watch
```
