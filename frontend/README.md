# GPX Helper Frontend

Svelte-based landing page for the GPX Helper tools.

## Prerequisites
- Node.js 18+
- npm

## Install dependencies
```sh
npm install
```

## Run the dev server
```sh
npm run dev
```

The app will be available at <http://localhost:4173>.

## Build for production
```sh
npm run build
```

## Run tests
```sh
npm run test
```

## Run tests in watch mode
```sh
npm run test:watch
```
